## Common Issues & Quick Fixes

| Issue | Fix |
|-------|-----|
| Camera not showing | Refresh page, check permissions, restart browser |
| Permission denied | Allow camera in browser settings |
| Face not recognized | Use same lighting/angle as registration |
| Backend not running | Check port 5001, run `python3 app.py` again |
| Frontend not running | Check port 3000, run `npm start` again |
